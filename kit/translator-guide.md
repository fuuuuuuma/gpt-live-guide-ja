# 🎁 リアルタイム翻訳サイトの使い方

配布サイト: https://fuuuuuuma.github.io/gpt-live-guide-ja/translate.html

## 何ができるか

マイクに向かって話すと、選んだ言語にその場で通訳し、音声でも読み上げてくれるWebサイトです。OpenAIのAPIキー（自分の利用枠を使うための鍵）を入れるだけで使えます。

このサイトは、OpenAIが提供している「Realtime API」という仕組みをブラウザから直接呼び出しています。間に中間サーバーを挟まない作りなので、話した内容や入力したAPIキーがこのサイトの運営者側に送られることはありません（通信はお使いのブラウザとOpenAIの間で直接行われます）。

ただし、OpenAIの公式な推奨では、通常のAPIキーは本来サーバー側で使うものとされています。ブラウザにキーを貼り付けるこのサイトの使い方は、自分の端末で使う個人利用向けの簡易な形です。下の「注意事項」を読んでから使ってください。

## 準備: OpenAIのAPIキーの取り方

1. https://platform.openai.com にアクセスし、OpenAIのアカウントでログインします
2. 左側のメニューから「API keys」を開きます
3. 「Create new secret key」をクリックして新しいキーを発行します
4. 表示されたキー（sk-から始まる文字列）をコピーして控えておきます
5. Realtime APIの利用には従量課金の登録が必要です。あらかじめOpenAIアカウントの支払い設定（Billing）で登録を済ませておいてください

## 使い方ステップ

1. 配布サイト（https://fuuuuuuma.github.io/gpt-live-guide-ja/translate.html）を開く
2. 控えておいたAPIキーを、キー入力欄に貼り付ける
3. 翻訳したい方向（どの言語からどの言語へ通訳するか）を選ぶ
4. 開始ボタンを押す
5. マイクに向かって話す。話した内容が選んだ言語に通訳され、音声で読み上げられる

## 注意事項

- APIキーは他人に共有しないでください。キーが漏れると、自分のOpenAIアカウントの利用枠を他人に使われてしまう可能性があります
- このサイト専用のキーを新しく発行し、OpenAIのBilling設定で利用上限（スペンドリミット）を設定してから使うことを強くおすすめします。使い終わったら platform.openai.com でそのキーを無効化（Revoke）すればより安心です
- 共有PCや公開Wi-Fiでは使わないでください（「このブラウザに保存する」は自分専用の端末でのみ使う）
- このサイトは運営者側のサーバーを持たず、キーは（保存を選んだ場合）お使いのブラウザ内にのみ保存されます
- 利用にはOpenAIのAPI利用料がかかります。事前にOpenAI側の料金体系を確認してください
- 周囲が騒がしい場所では、音声の聞き取り精度が落ちることがあります。できるだけ静かな環境での利用をおすすめします
